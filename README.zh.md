## **1. 产品愿景与概述**

**产品名称：** Anamnesis  
**核心理念：** 一个纯粹的、用户主权至上的 Web3 永久存储保险箱。其名称源于柏拉图哲学中的“灵魂回忆说”，寓意此工具帮助用户的数字灵魂找回并永久守护那些本应永恒的记忆与数据。  
**产品定位：** 一个纯前端的应用，作为一个安全的“账户与数据金库”。它允许用户管理多链账户，并自由选择将文件**永久刻录**在 Arweave 公链上，或**私密封存**在 Irys 存储网络。

## **2. 核心功能**

### **2.1 多链账户管理**

- **广泛支持**：支持管理 **Ethereum, Bitcoin (Taproot), Solana, Sui, 以及 Arweave** 的账户。
- **隐私金库**：所有账户通过**主密码**在本地加密。使用 PBKDF2 算法进行密钥派生，XChaCha20-Poly1305 进行加密。
- **自动识别**：导入私钥、助记词或 Arweave JWK，系统会自动识别对应的链类型。
- **助记词支持**：支持为主要区块链生成标准的 12 位 BIP39 助记词并安全存储。

### **2.2 隐私优先的存储**

- **端到端加密 (E2EE)**：文件在离开浏览器前，使用主密码派生的密钥在本地完成加密。
- **双存储协议支持**：
  - **Arweave 原生**：通过原生 AR 交易实现真正的永久存储。
  - **Irys 网络**：快速、可扩展的存储，支持多币种支付（ETH, MATIC 等）。
- **元数据保护**：文件名等元数据在上传前均会被加密，确保链上数据的完全隐私。

### **2.3 安全数据检索**

- **统一控制面板**：在一个界面查看所有永久和私密的上传记录。
- **链上同步**：直接从区块链网关同步交易历史。
- **本地解密**：一键解密并下载文件。私钥绝不接触网络。

## **3. 安全架构**

- **零知识后端**：无服务器，无数据库。你的数据完全由你控制。
- **主密码系统**：本地“隐私层”保护已保存的密钥和加密文件。
- **PBKDF2 派生**：高迭代次数的密钥派生，有效防御暴力破解。
- **分区隔离**：数据基于主密码进行“金库”化隔离，支持创建隐藏分区。
- **敏感信息安全查看**：查看私钥或助记词时需要重新进行密码验证。

## **4. 技术栈**

- **前端**: React + Vite + Tailwind CSS + Shadcn/UI
- **存储**: `arweave-js`, `@irys/sdk`
- **加密**: `libsodium-wrappers`, Web Crypto API (PBKDF2)
- **数据库**: `dexie` (IndexedDB 封装)
- **区块链相关**: `ethers`, `bitcoinjs-lib`, `@solana/web3.js`, `@mysten/sui`, `bip39`, `bip32`
- **多语言**: `i18next`
