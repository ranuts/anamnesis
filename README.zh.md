## **产品简介**

**Anamnesis** 是一个以“用户主权”为核心的 **账户与永久文件金库**。  
名字源自柏拉图的“回忆说（Anamnesis）”，寓意帮助你保存、找回并长期守护那些不该丢失的数字记忆与重要资料。

**在线地址**：https://ranuts.github.io/anamnesis/

## **你可以用它做什么**

- **管理多链账户**：把常用钱包统一整理（如 Ethereum、Bitcoin、Solana、Sui、Arweave）。
- **长期保存文件**：在“公开永久存储”和“私密封存存储”之间自由选择。
- **统一查看记录**：在一个面板里管理上传与相关记录，清晰可追溯。
- **随时取回**：需要时可下载并解密私密文件。

## **隐私与安全原则**

- **数据归你所有**：敏感信息在本地被保护，Anamnesis 不会收集或托管。
- **尽量去中心化依赖**：减少对中心化保管与信任的要求。
- **敏感操作二次确认**：在展示高度敏感信息前进行额外确认。

## **适用场景**

- **个人长期档案**：票据、证书、照片、重要文档。
- **创作者存证**：作品发布与时间证明，方便长期引用。
- **团队交接**：关键资料长期保存，按需取回与查验。
